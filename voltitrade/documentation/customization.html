Customization

Every visual aspect of Vaultex is controlled through CSS variables and a handful of swappable files — no component markup needs to change.

Colors

Edit the :root block at the top of assets/css/style.css:

--mint: #00D9A3;   /* primary accent */
--gold: #F2B84B;   /* highlights */
--ink:  #0A0E1A;   /* background */

Alternate brand presets

Three ready-made accent palettes ship in assets/css/themes.css. Apply one with an attribute on the <html> tag:

<html data-theme="dark" data-brand="violet">

Logo & brand name

Replace the <span class="mark">V</span> letter-mark with your own <img> tag, and update the Vaultex text — it appears as plain text in every page, not baked into an image, so a find-and-replace across all .html files works.

Fonts

Swap the Google Fonts <link> tag in each page's <head>, then update --font-display, --font-body, and --font-mono in style.css.
